# Sprint Planning

This document tracks the MVP build plan.

## MVP Build Order

1. Finalize documentation and engineering standards.
2. Scaffold FastAPI backend and React frontend.
3. Add SQLite connection and SQLModel base models.
4. Build startup CRUD API.
5. Build React app shell.
6. Build startup list with TanStack Table.
7. Build startup profile view.
8. Add founder, funding, and signal sections.
9. Add thesis workspace backed by Markdown files.
10. Add scorecard framework.
11. Add dashboard summaries and charts.
12. Add pytest backend coverage.
13. Add Playwright smoke tests.
14. Add seed examples for Indian AI startups.
15. Add export/import planning for JSON, CSV, and Markdown.

## Sprint 0

Goal: Establish project foundation.

Candidate tasks:

- Complete project documentation.
- Confirm backend and frontend scaffolding strategy.
- Define initial SQLModel entities.
- Define initial API route map.
- Define frontend route map.

## Sprint 1

Goal: Create the first working startup database loop.

Candidate tasks:

- Scaffold backend.
- Scaffold frontend.
- Create SQLite database setup.
- Implement startup create/list/read/update.
- Build startup list and startup profile basics.
- Add one backend pytest path.
- Add one Playwright smoke test.

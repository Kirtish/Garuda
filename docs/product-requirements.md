# Product Requirements

## MVP Goal

Build a local research workspace where an individual investor can maintain a structured database of Indian AI startups, evaluate them with a repeatable scoring framework, and write durable investment theses.

## Core User Stories

- As an investor, I can add and manage Indian AI startup profiles.
- As an investor, I can track founders and founder research notes.
- As an investor, I can record funding rounds with source links.
- As an investor, I can capture signals such as hiring, launches, partnerships, customer wins, media mentions, and funding activity.
- As an investor, I can write an investment thesis for each company.
- As an investor, I can score companies using a structured framework.
- As an investor, I can manage company status across watchlist, researching, shortlisted, passed, and invested.
- As an investor, I can view dashboards and filtered tables to prioritize research.

## MVP Scope

- Startup database
- Founder research
- Funding tracker
- Signal tracker
- Investment thesis workspace
- Structured scorecard
- Portfolio and watchlist views
- Dashboard summary

## Non-Goals

- Automated web scraping
- Real-time distributed monitoring
- User accounts
- Team collaboration
- Cloud sync
- Paid data integrations
- Advanced predictive recommendations
- Kubernetes, microservices, Redis, Kafka, RabbitMQ, or cloud deployment

## First Milestone

The first usable milestone is:

> Add 20 Indian AI startups, research founders, record signals, score each company, and maintain a clear investment thesis for the watchlist.

## MVP Screens

- Dashboard
- Startup list
- Startup profile
- Add/edit startup
- Founder section
- Funding section
- Signal section
